# dangibbs.uk

Dan Gibbs's blog and personal website built with 
[Eleventy](https://github.com/11ty/eleventy).

## Build

```
npm run build
```

## Develop

```
APP_URL="http://localhost:8080" npm run serve
```

## Test

Run all tests:

```
npm run test
```

TDD integration tests only:

```
npm run test-integration
```
